# Frontend

Dashboard user interface built with React/Next.js.

## Features

- Unified inbox interface
- Lead and job pipeline visualization
- AI agent chat interface
- Estimate management
- Scheduling interface
- Customer profiles
- Activity logs
- Notifications

## Getting Started

```bash
cd src/frontend
npm install
npm run dev
```

## Structure

- `components/` - Reusable UI components
- `pages/` - Next.js pages
- `hooks/` - Custom React hooks
- `context/` - React context providers
- `utils/` - Utility functions
- `styles/` - CSS/styling files
